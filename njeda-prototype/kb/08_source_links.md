## NJEDA source links (for KB expansion)

Use these to add new programs to `03_program_catalog.jsonl` and to refresh existing ones.

### Primary site
- https://www.njeda.gov/

### Common hubs / patterns to search
- Pages containing: `program`, `financing`, `incentive`, `tax credit`, `grant`, `loan`, `procurement`, `vendor`, `RFP`, `bid`, `training`, `workforce`
- Sections in navigation: Financing and Incentives, Strategic Industry Support, Real Estate, Public Information

### How to add a program (quick workflow)
1) Locate the canonical NJEDA program page URL.
2) Extract: title, summary, who it’s for, benefit type, eligibility bullets, how to apply.
3) Create/append a JSON line in `03_program_catalog.jsonl`.
4) Add tags and `last_verified`.
5) Re-upload updated KB file(s) to Lyzr Studio.
